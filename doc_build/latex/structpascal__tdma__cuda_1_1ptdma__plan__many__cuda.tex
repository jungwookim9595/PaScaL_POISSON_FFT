\doxysubsection{Detailed Description}
Execution plan for many tridiagonal systems of equations. 

It uses MPI\+\_\+\+Ialltoall function to distribute the modified tridiagonal systems to MPI processes and build the reduced tridiagonal systems of equations. Currently it supports the equal size of domains. It does not use derived datatypes.
