\doxysubsection{Detailed Description}
Execution plan for many tridiagonal systems of equations. 

It uses MPI\+\_\+\+Ialltoallw function to distribute the modified tridiagonal systems to MPI processes and build the reduced tridiagonal systems of equations. Derived datatypes are defined and used to eliminate the cost of data packing and unpacking.
